# FlipBook
Module FlipBook for Wordpress.

## Version
1.0.1

## Usage
Activate the plugin, then add the shortcode below to a page:

```
[flipbook title="Living Design" images="url1,url2,url3,url4"]
```

Or render a PDF with streaming support for large files:

```
[flipbook title="Catalogue" pdf="https://example.com/catalogue.pdf"]
```

If no images or pdf are supplied, the plugin falls back to the bundled sample pages.
